DDOS Attack

The subject of our second activity is DDOS attack and of course we will use kali linux for this, first we log in to kali linux, then we start exercise with the msfconsole command, so what is msfconsole? The msfconsole is the command-line interface for Metasploit, a popular penetration testing framework used for exploiting vulnerabilities, conducting security assessments, and performing other related tasks. It provides a powerful and flexible environment for interacting with the Metasploit Framework.

This is exactly the screen we need to see. When we see this symbol, we know we are on the right track.

Then we use show options, which is one of the basic commands of msfconsole, and see the options.

Finally, we set the setREPORT and setRHOST, enter our IP and run it with the run command..
